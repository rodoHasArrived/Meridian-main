# Meridian Brand Assets

Meridian now includes a small first-party brand pack for the workstation shell, packaging, and documentation surfaces.

## Concept

The mark combines three ideas already present in the product direction:

- a vertical meridian beam for orientation and precision
- a rising signal path for research, trading, and operational flow
- a restrained horizon curve to keep the identity calm and institutional instead of promotional

The palette stays inside the existing WPF shell language: deep navy foundations with controlled blue, cyan, and green accents.

## Usage Guidance

- Use the mark or tile for application chrome, installers, and shell surfaces.
- Use the wordmark when the full product name needs to be explicit.
- Use the hero visual as supporting atmosphere only; it should stay behind operational content rather than compete with it.
- Keep the brand treatment secondary to trust cues, environment labels, and operator context on real workstation pages.
