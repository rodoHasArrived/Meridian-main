# Prompt Templates

This directory is the navigation index for **reusable AI prompt templates** used by GitHub Copilot
Chat and other AI tools. The actual prompt files live in `.github/prompts/` where Copilot can
discover them.

---

## Available Prompts

## Using Prompts in Copilot Chat

Reference a prompt file directly in Copilot Chat:

```
@workspace /fix-build-errors
```

Or use them as a starting template and customize for your specific task.

---

## Auto-Generation

The [`prompt-generation.yml`](https://github.com/rodoHasArrived/Meridian-main/blob/main/.github/workflows/prompt-generation.yml) workflow
automatically generates targeted prompts from CI/CD failures:

1. Analyzes failed workflow runs (test-matrix, code-quality, security, benchmarks)
2. Classifies failures into categories
3. Generates `.prompt.yml` files with failure context
4. Creates a PR with the generated prompts (optional)

Trigger manually: **Actions → Prompt Generation → Run workflow**

## Adding New Prompts

1. Create a new `.prompt.yml` file in `.github/prompts/`
2. Follow the YAML format used by existing prompts
3. Add an entry to [`.github/prompts/README.md`](https://github.com/rodoHasArrived/Meridian-main/blob/main/.github/prompts/README.md)
4. Add an entry to this README
