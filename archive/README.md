# Meridian Archive

Historical and non-active project material lives here instead of under `docs/`.

## Structure

- `docs/` - Archived documentation, organized by document type
- `code/` - Placeholder for any future retired code snapshots or extracted non-active source

## Notes

- Active documentation remains under `docs/`.
- Active source code remains under `src/`, `tests/`, `scripts/`, and related project folders.
- Only clearly historical or superseded material should be moved here.
