# Archived Code

This folder is reserved for code that is clearly retired, orphaned, or preserved only for historical reference.

No active source from `src/`, `tests/`, or runtime project folders should be moved here unless it has been explicitly retired from the build and reviewed for dependency impact.

## Current Subfolders

- `obj-codex/` - Mirrored archive of stray `obj-codex` directories that were removed from active `src/` and `tests/` project trees.
