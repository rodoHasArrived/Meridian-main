System notes03 / 04

Palette
Base is deep navy #08101A. Surface stack: card #0D1722 → panel #101C2B → soft #162334 → raise #1B2A3C. A single signal primary #2AB2D4 carries CTAs, active nav, focus rings, chart lines, and selected-row emphasis. Semantic tones remain structural: healthy, observe, critical, paper, research, pending, inactive.
Typography
Three families, strict roles. Space Grotesk 600 for display titles. IBM Plex Sans 400/500 for UI prose and labels. IBM Plex Mono — tabular — for every number, identifier, timestamp, row count, price, and KPI. Letter spacing stays at 0 except mono eyebrows and table headers.
Geometry
Radii step 3 / 4 / 6 / 8 / 10 px. Default border #1F344C; high-contrast separators and active frames use #2A4566. Shadows are shallow with faint inset highlights; no large glow effects. Uploaded custody/reporting references favor crisp rows and narrow controls.
Workbenches
Use .workstation-frame, .workstation-masthead, .operator-rail, .toolbar-strip, .metric-strip, .split-workbench, .dense-table, and .entity-summary for image-inspired operator pages. Summary metrics must connect to chart/table/detail evidence.
