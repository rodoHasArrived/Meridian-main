TradingExecution readiness
Layout
Masthead, operator rail, readiness KPI strip, split workbench with orders table and selected-session inspector.
Primary evidence
Session state, buying power, replay verification, open orders, fills, guardrail results, broker sync.
Operator actions
Create paper session, verify replay, submit order, cancel order, assign review, promote when gates pass.
States
Paper, live, pending, stale replay, critical guardrail, disabled promotion, selected order.
Primitives
workstation-frame, metric-strip, dense-table, status-window, row-detail-panel, state-matrix-badge.
Empty/error
No paper sessions active. Create one above. Replay stale. Rerun verification before accepting readiness.
PortfolioExposure review
Layout
KPI row above chart/table evidence, holdings table on the left, exposure and factor inspector on the right.
Primary evidence
Market value, exposure, P&L, drawdown, top contributors, concentration, benchmark deltas.
Operator actions
Filter account, inspect holding, export exposure, open Security Master, compare benchmark.
States
Healthy exposure, observe concentration, critical breach, stale marks, selected holding, empty account.
Primitives
metric-strip, chart-standard-card, dense-table, entity-summary, row-detail-panel.
Empty/error
No holdings for selected account. Clear filters or import positions. Mark refresh failed with reason.
